# BigTime API

## Prerequisites
- [Node 6.x & NPM](https://nodejs.org) (or `brew install node`)

## Setup
1. Run `npm i`

## Running
1. Run `npm run setup` to obtain an API token.
1. Run `npm run app` to start the main app.
1. Run `npm run sample` to start the data sampling and submittal app.

## References
